Instructions

  1. Enable or disable automatically setting the volume.
  2. Enable or disable starting the app at login.
  3. Set the target value you’d like the volume to be set to.
  4. Quit the app—this stops the volume from automatically being set.
  5. Closes the window—the app will continue running in the background.
